In progress: Lauren Bell Honors Thesis, expected completion Fall 2026

R Shiny tool for gene set enrichment analysis using International
Mouse Phenotyping Consortium mouse phenotype data. Supports multiple 
null distribution options (blitzGSEA, npGSEA, and gene set permutation) 
for more flexible and less computationally taxing enrichment testing.

Built for the Morgun-Shulzhenko Lab @ Oregon State University.
